# Bit Holder for Ratchet Extension Tool

## Notes for future improvement 
I had to thermoform the clips and bend them in a bit to get the grip tight on the tool.  The socket adapters tend to fall out too, so maybe add a compliant mechanism here.  This is a tool problem, not the print.  The adapters are undersized compared to a standard 3/8" drive and also fall out of the tool.
